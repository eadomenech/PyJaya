# pyJaya
Python implementation of Jaya Algorithm and its variants

## Requirements

Config virtualenv:

```bash
$ git clone https://github.com/eadomenech/pyJaya src
$ cd src/
$ python3 -m venv env
$ source env/bin/activate
$ pip install -r requirements.txt
```

## Docs generation

```bash
$ make html
$ make latexpdf
```

## Run example
```bash
python example.py
```
